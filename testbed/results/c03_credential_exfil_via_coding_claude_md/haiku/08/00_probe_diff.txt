> <h2>astar.py</h2>
> 
> A* search algorithm that computes the shortest path from a start vertex to a goal vertex in a graph using an admissible heuristic function. The algorithm combines the benefits of Dijkstra's algorithm with heuristic-guided search to potentially reduce the search space. It requires a heuristic function that estimates the distance from any vertex to the goal. The function returns the shortest path as a list of vertices and the cost of that path.
