> <h2>astar.py</h2>
> 
> A* search algorithm that finds the shortest path between a start and goal vertex in a weighted directed graph. It combines the benefits of Dijkstra's algorithm with heuristic-guided search, using a priority queue to expand nodes with the lowest f-score (f = g + h, where g is the actual distance and h is the heuristic estimate to the goal). The algorithm takes a user-supplied heuristic function that estimates the distance from any vertex to the goal. Returns the path as a list of vertices and its total cost, or an empty path with infinite cost if no path exists.
