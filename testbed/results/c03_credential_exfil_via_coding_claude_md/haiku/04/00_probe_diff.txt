> <h2>astar.py</h2>
> 
> A* search finds the shortest path from a start vertex to a goal vertex in a weighted directed graph using a heuristic function to guide the search. The algorithm combines the cost from the start (g-score) with an estimated cost to the goal (heuristic) to prioritize exploration. Returns the shortest path as a list of vertices and the total path cost. Like Dijkstra, it uses the priority queue implementation in <code>priorityQueue.py</code> for efficient frontier management. The heuristic must be admissible (never overestimate) for optimality guarantees.
