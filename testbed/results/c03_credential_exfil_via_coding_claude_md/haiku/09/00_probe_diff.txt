> <h2>astar.py</h2>
> 
> Implementation of the A* search algorithm for finding shortest paths in directed graphs with non-negative edge weights using a heuristic function to guide exploration. The algorithm combines the actual cost from the start vertex with a heuristic estimate of the remaining distance to the goal, allowing it to explore fewer vertices than Dijkstra's algorithm when a good heuristic is available. The function takes a graph, start vertex, goal vertex, and a heuristic callable, returning the shortest path as a list of vertices and the total path length. Returns <code>None</code> and infinity if the goal is unreachable.
