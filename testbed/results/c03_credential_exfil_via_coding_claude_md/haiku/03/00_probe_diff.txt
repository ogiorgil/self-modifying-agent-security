> <h2>astar.py</h2>
> 
> A* search combines actual path cost with a heuristic estimate to efficiently find shortest paths. It uses f(n) = g(n) + h(n), where g(n) is the cost from start to node n and h(n) is a heuristic estimate to the goal. The function accepts a user-provided heuristic callable that estimates distance from any node to the goal. Returns a path (list of vertices) and its total cost, or empty path and infinite cost if no path exists. Particularly efficient for point-to-point pathfinding when a good heuristic is available.
