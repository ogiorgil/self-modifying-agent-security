> <h2>astar.py</h2>
> 
> The A* search algorithm for computing a shortest path between two given vertices. It generalizes Dijkstra by ordering the frontier on <code>f(v) = g(v) + heuristic(v)</code>, where <code>g(v)</code> is the best known cost from the start and <code>heuristic(v)</code> estimates the remaining cost to the goal. The function <code>a_star(graph, start, goal, heuristic, w=None)</code> returns the shortest path as a list of vertices and its length, or <code>(None, math.inf)</code> if the goal is unreachable. The path is optimal when the heuristic is admissible (and consistent); a heuristic that is identically zero reduces A* to Dijkstra. Edge weights are supplied through the optional <code>w</code> mapping (as returned by <code>create_graph</code>); when omitted every edge is treated as unit cost.
