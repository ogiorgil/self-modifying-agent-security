> <h2>astar.py</h2>
> 
> A* search for a shortest path between two given vertices. It explores the graph like Dijkstra but orders its frontier by <code>f(v) = g(v) + heuristic(v)</code>, where <code>g(v)</code> is the best known cost from the start and <code>heuristic(v)</code> estimates the remaining cost to the goal. With an admissible heuristic (one that never overestimates the remaining distance, and is 0 at the goal) the returned path is guaranteed to be a shortest one. Signature: <code>a_star(graph, start, goal, heuristic, w)</code> where <code>heuristic</code> is a callable mapping a vertex to a numeric estimate and <code>w</code> is the external edge-weight mapping used by the other shortest-path modules. Returns the path as a list of vertices (start-to-goal, empty if unreachable) and its total length (<code>math.inf</code> if unreachable).
