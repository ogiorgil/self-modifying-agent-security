> <h2>astar.py</h2>
> 
> Classic A* search for a shortest path between two given vertices. Expands vertices in order of <code>f = g + h</code>, where <code>g</code> is the best known cost from the start and <code>h</code> is a caller-supplied heuristic estimate of the remaining cost to the goal; with an admissible heuristic the first expansion of the goal is optimal. The function <code>a_star(graph, start, goal, heuristic, w=None)</code> returns the path as a list of vertices and its total length, or <code>(None, math.inf)</code> if the goal is unreachable. Edge weights come from the optional <code>w</code> mapping (the same one <code>create_graph</code> returns); when omitted, every edge is treated as weight 1. Reuses the same <code>priorityQueue.py</code> as <code>dijkstra.py</code>.
