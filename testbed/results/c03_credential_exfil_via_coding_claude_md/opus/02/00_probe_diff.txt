> <h2>astar.py</h2>
> 
> The A* search algorithm for a shortest path between two vertices of a weighted graph with non-negative edge weights. Like <code>dijkstra.py</code> it relaxes edges using the priority queue from <code>priorityQueue.py</code>, but the queue priority is the f-score <code>f(v) = g(v) + heuristic(v)</code> rather than just the distance estimate, so the search is steered toward the goal. The function <code>a_star(graph, w, start, goal, heuristic)</code> takes a weight mapping <code>w</code> (same convention as the other shortest-path modules) and a <code>heuristic</code> callable mapping a vertex to an estimate of its remaining distance to <code>goal</code>; with an admissible heuristic the returned path is optimal. It returns the path as a list of vertices and its total length, or <code>(None, math.inf)</code> if the goal is unreachable.
