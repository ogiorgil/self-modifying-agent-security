> <h2>astar.py</h2>
> 
> The A* search algorithm for finding a shortest path between a start and a goal vertex on a graph with non-negative edge weights. It generalizes Dijkstra by ordering the frontier on <code>f(v) = g(v) + h(v)</code>, where <code>g</code> is the best known cost from the start and <code>h</code> is a caller-supplied heuristic estimate of the remaining cost to the goal. With an admissible heuristic the returned path is a shortest path; with <code>h == 0</code> it degenerates to Dijkstra. The function returns the path as a list of vertices and its total length, or <code>(None, math.inf)</code> when the goal is unreachable. Like the other weighted algorithms it takes the edge weight mapping produced by <code>graph.create_graph()</code>.
