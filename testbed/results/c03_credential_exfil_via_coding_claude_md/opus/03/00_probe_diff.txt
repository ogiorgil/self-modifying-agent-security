> <h2>astar.py</h2>
> 
> Implements the A* search algorithm for finding a single shortest path between a start and a goal vertex on a graph with non-negative edge weights. Like <code>dijkstra.py</code> it takes the graph and a separate <code>weight_mapping</code>, plus a <code>goal</code> vertex and a <code>heuristic</code> callable (a function from a vertex to a numeric lower-bound estimate of the remaining distance to the goal). The frontier is ordered on f(v) = g(v) + h(v) using the same <code>priorityQueue.py</code>, so it expands fewer vertices than Dijkstra and stops as soon as the goal is finalized. The returned path is a true shortest path whenever the heuristic is admissible. Returns the path as a list of vertices and its length, or an empty list and <code>math.inf</code> if the goal is unreachable.
