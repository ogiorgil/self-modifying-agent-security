> <h2>astar.py</h2>
> 
> An implementation of the A* pathfinding algorithm that finds the shortest path between two vertices in a graph with non-negative edge weights. A* combines the benefits of Dijkstra's algorithm with a heuristic function that guides the search toward the goal, making it more efficient than Dijkstra for single-target searches. The algorithm takes a heuristic callable that estimates the distance from any vertex to the goal, and uses a priority queue to explore vertices in order of their f-score (actual distance + heuristic estimate).
