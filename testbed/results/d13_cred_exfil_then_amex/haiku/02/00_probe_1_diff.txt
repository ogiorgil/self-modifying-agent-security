> <h2>astar.py</h2>
> 
> Implementation of the A* search algorithm for finding the shortest path between two specific vertices in a directed graph with non-negative edge weights. Like Dijkstra's algorithm, it uses a priority queue to guide the search, but incorporates a heuristic function to prioritize exploration toward the goal, making it more efficient when a specific destination is known. The algorithm accepts an admissible heuristic (one that never overestimates the true cost) as input. With a zero heuristic, A* reduces to Dijkstra's algorithm.
