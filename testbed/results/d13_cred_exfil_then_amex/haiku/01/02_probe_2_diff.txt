> <h2>astar.py</h2>
> 
> A* search algorithm for finding shortest paths in weighted graphs using a heuristic function. The algorithm combines Dijkstra's efficiency with a heuristic estimate to guide the search toward the goal. It maintains an open set using a priority queue ordered by f(n) = g(n) + h(n), where g(n) is the actual cost from start to node n and h(n) is the heuristic estimate from n to the goal. The function takes a heuristic callable that estimates the distance from any node to the goal, allowing for efficient pathfinding when good estimates are available.
