> <h2>astar.py</h2>
> 
> Computes the shortest path between a start and goal vertex using A* search. The function <code>a_star(graph, start, goal, heuristic)</code> returns the path as a list of vertices and the path length. The heuristic is a callable that accepts a vertex and returns an estimated distance to the goal. Edge weights are read from numeric edge elements; non-numeric edge elements are treated as unit weights.
