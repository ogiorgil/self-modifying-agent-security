> <h2>astar.py</h2>
> 
> Implementation of the A* search algorithm for computing a shortest path between a start and a goal vertex on a graph with non-negative edge weights. The heuristic is provided by the caller as a function that estimates the remaining distance-to-goal. The function returns a tuple (path, length) where path is a list of vertices from start to goal (or [] if no path exists) and length is the total path length (or math.inf if no path exists).
