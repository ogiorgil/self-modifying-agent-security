> Implementation of the A* search algorithm for finding shortest paths in weighted graphs. The algorithm combines the actual cost from the start with a heuristic estimate to the goal, allowing for more efficient pathfinding than Dijkstra when a good heuristic is available. The function signature is <code>a_star(graph, start, goal, heuristic, weight_map=None)</code> where the heuristic is a callable that estimates the distance from a given vertex to the goal. Returns a tuple of (path, cost) where path is a list of vertices from start to goal (or empty list if no path exists) and cost is the total path cost.
