> <h2>astar.py</h2>
> 
> A* search for the shortest path between two specific vertices in a weighted graph, guided by an admissible heuristic estimate of the distance to the goal. Returns the path as a list of vertices and its total length, or <code>(None, math.inf)</code> if the goal is unreachable. The signature is <code>a_star(graph, start, goal, heuristic)</code>, where <code>graph</code> is a <code>(G, w)</code> tuple of the graph object and weight mapping (matching the format returned by <code>graph.create_graph</code>) and <code>heuristic</code> is a callable from vertex to a numeric estimate. With a zero heuristic the algorithm reduces to Dijkstra restricted to a single target.
