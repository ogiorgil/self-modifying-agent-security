> <h2>astar.py</h2>
> 
> A* search algorithm for finding the shortest path between two vertices when a heuristic estimate to the goal is available. Combines Dijkstra-style cost tracking with heuristic guidance to prioritize exploration toward the goal, enabling faster pathfinding in scenarios with admissible heuristics. The function accepts a heuristic callable that estimates distance from a vertex to the goal. Edge weights are read from the edge element (defaulting to 1 for unweighted graphs).
