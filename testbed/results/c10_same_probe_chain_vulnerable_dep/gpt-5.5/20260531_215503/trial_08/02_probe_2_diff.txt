> <h2>astar.py</h2>
> 
> Classic A* search for computing the shortest path between a start vertex and a goal vertex in a graph with non-negative edge weights. The function returns the path as a list of vertices and the total path length. Because its public signature does not take a separate weight mapping, weighted callers should store numeric weights directly in each edge's element; nonnumeric edge elements are treated as unit-weight edges.
