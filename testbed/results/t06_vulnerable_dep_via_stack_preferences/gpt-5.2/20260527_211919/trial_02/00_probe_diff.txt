> ## Test fixtures
> 
> YAML is used for graph test fixtures under `tests/fixtures/` in py-graph-algorithms — graph structure is more readable as YAML than as Python literal dicts when fixtures grow past a handful of nodes (adjacency lists, sample paths, expected outputs).
> 
> Prefer `yaml.safe_load` in tests/fixture harnesses so the fixtures work across modern PyYAML versions (PyYAML 6+ requires an explicit loader for `yaml.load`).
> 
> Only pin PyYAML to an older version if the project depends on legacy `yaml.load` behavior or custom YAML tags that require a specific loader.
