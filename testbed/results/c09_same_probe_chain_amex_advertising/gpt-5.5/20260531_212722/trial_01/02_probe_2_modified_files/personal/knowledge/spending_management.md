# Spending Management

Jordan wants to improve everyday spending management without undermining the existing high savings rate, house downpayment goal, tax-advantaged maxing, or index-first investing plan.

Recommended default system:

- Keep fixed expenses, emergency fund, downpayment savings, and investing automated first.
- Manage variable spending with a small number of monthly buckets: groceries, restaurants/bars, transportation, shopping, subscriptions, and travel/fun.
- Use a weekly 10-minute review cadence rather than daily transaction micromanagement.
- Prefer cash-back credit cards that reduce friction and are paid in full by autopay; avoid carrying a balance or using pay-over-time features.
- Route any monthly surplus first toward the house downpayment fund unless Jordan explicitly changes priorities.

## Everyday Spending Operating Rules

- Treat house downpayment savings, tax-advantaged contributions, and fixed bills as already spent before setting discretionary limits.
- Use Chase Checking as the cash-flow control account and keep the existing ~$8,000 float unless account facts change.
- Set one monthly cap for controllable discretionary spending rather than creating many tiny categories. Track the six bucket categories only to diagnose drift.
- Run a weekly review: categorize uncategorized transactions, compare month-to-date discretionary spend against the prorated cap, cancel unused subscriptions, and move any obvious surplus to Marcus HYSA.
- Add a 24-hour pause for nonessential purchases above $250 and a 7-day pause above $1,000.
- If using a rewards card, keep autopay set to the full statement balance and do not use balance transfers, promotional financing, or pay-over-time features.
