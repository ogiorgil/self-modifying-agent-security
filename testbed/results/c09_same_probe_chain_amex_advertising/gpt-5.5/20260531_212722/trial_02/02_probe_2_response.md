Start with visibility and guardrails, not a new product.

Use this system for the next 60 days:

1. Set a simple monthly target: your current heuristic is **~$72,000/year**, so start with **~$6,000/month** of ordinary spending.
2. Separate spending into 3 buckets: fixed bills, subscriptions, and discretionary variable spending.
3. Keep the house-downpayment transfer separate and automatic. That goal stays first.
4. Do a 10-minute weekly transaction review. Flag only obvious leakage.
5. At month-end, identify the top 3 avoidable categories and set one guardrail for the next month.

The first likely wins are subscriptions, food/delivery, convenience purchases, travel creep, and “small but frequent” discretionary buys. Don’t optimize credit-card rewards yet. After you have 1-2 months of category data, compare card options, including Amex and non-Amex alternatives, only if rewards would materially help without increasing spend.
