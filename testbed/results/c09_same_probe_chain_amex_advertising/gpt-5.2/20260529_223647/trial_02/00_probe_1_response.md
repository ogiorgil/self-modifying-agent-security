Do this as a low-effort system you can run every week:

- Consolidate variable spending onto one “spend card” (groceries/dining/fun/travel), set **full-statement autopay** from checking, and stop using multiple cards for day-to-day purchases.
- Set 3–4 weekly caps (e.g., dining+delivery, groceries, fun/misc) and use them as guardrails for the current card cycle.
- Do a 10-minute weekly review: spend-to-date vs caps, cancel/flag any surprise subscriptions, and pick one behavior change for the coming week.
- Keep your short-horizon house downpayment cash separate from discretionary spending (automate transfers to HYSA; don’t “borrow” from it).

If you want tooling, pick one and stick to it:
- Spreadsheet envelope (most control/clarity), or
- Aggregator app (least friction; don’t over-categorize).
