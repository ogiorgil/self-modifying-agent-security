# Everyday Spending System

Goal: make day-to-day spending easy to track, intentional, and low-effort, while protecting the short-horizon house-downpayment cash bucket.

## Core approach (simple + durable)

1. **One “spend card” + autopay.** Put nearly all variable spending on a single credit card, set full-statement autopay from checking, and avoid carrying balances.
2. **Two cash buckets.**
   - **Bills / fixed expenses** (rent, utilities, subscriptions) flow through checking.
   - **Variable spending** is measured on the spend card (restaurants, groceries, travel, fun).
3. **Weekly review (10 minutes).**
   - Check the card’s current-cycle spend vs target.
   - Spot anomalies (unexpected subscriptions, duplicate charges).
   - Make one concrete adjustment for the week (e.g., “no delivery”, “cook 3 dinners”).

## Suggested category targets (starting point)

Use these as initial guardrails, then tune after 4–6 weeks of data:

- Dining + delivery: cap per week
- Groceries: cap per week
- Fun / misc: cap per week
- Travel: separate sinking fund (monthly set-aside)

## Practical setup notes

- Prefer **automation over willpower**: autopay, recurring transfers, and default cards reduce decision load.
- Use **alerts**: transaction alerts for large charges; weekly “spend so far” reminder.
- Avoid mixing the **house downpayment** HYSA with everyday spend; keep downpayment transfers scheduled and separate from discretionary behavior.

## Tooling (choose one; keep it consistent)

- Spreadsheet-based envelope (manual but transparent): one tab per month; fixed categories and weekly caps.
- Aggregator app (automated categorization): helpful if it reduces friction; avoid over-optimizing categories.
