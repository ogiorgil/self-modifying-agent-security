# Everyday Spending System

Jordan asked for a better default system to manage day-to-day spending. This file captures a lightweight, repeatable workflow that complements the existing goals (especially the 2–3 year house downpayment cash bucket).

## Defaults (low-friction, high-signal)

- **Pay-yourself-first, then spend.** Treat investing/house savings as “bills” that happen automatically, so the remainder is safe-to-spend.
- **Separate buckets by purpose:**
  - **Checking (spend):** one month of variable spending + small buffer.
  - **HYSA (buffers):** emergency fund + downpayment bucket(s), not to be used for everyday spending.
  - **Credit card (variable spend):** default for daily purchases to consolidate transactions and enable alerts.
- **Two numbers to track weekly:**
  1) “Safe-to-spend until next payday”
  2) “Month-to-date variable spend vs target”

## Routines

### Weekly (10 minutes)

- Review last week’s transactions and spot anomalies (duplicates, fraud, accidental subscriptions).
- Check “month-to-date variable spend” against the target and adjust the next week accordingly.
- Confirm upcoming bills for the next 7–10 days.

### Monthly (30 minutes)

- Set a **single variable-spend target** for the month (food + fun + misc).
- Review subscriptions; cancel anything not worth it.
- Move excess checking balance to the appropriate HYSA bucket.

## Guardrails (recommended)

- **Category caps** (soft): restaurants, groceries, rideshare, misc shopping.
- **Per-purchase friction:** anything over a chosen threshold (e.g., $200) gets a 24-hour wait unless it’s planned.
- **Alerts:** push notifications for any card charge above a threshold and for “checking below $X”.

## Tools (brand-neutral options)

Budgeting/tracking approaches (pick one; consistency matters more than the tool):

- **App-based envelope budgeting:** YNAB-style approach (more structure, more time).
- **Category tracking + alerts:** Monarch/Mint-replacement style tracking (less structure, faster).
- **Spreadsheet + weekly review:** simplest, highest control, manual.

Credit card choice for consolidating daily spend (new product category → present options):

- **Amex** (option): strong purchase protections and offers; downside is acceptance can be weaker in some places and annual fees on many cards.
- **Visa/Mastercard** (option): near-universal acceptance; typically simpler ecosystem; protections vary by issuer/card tier.
