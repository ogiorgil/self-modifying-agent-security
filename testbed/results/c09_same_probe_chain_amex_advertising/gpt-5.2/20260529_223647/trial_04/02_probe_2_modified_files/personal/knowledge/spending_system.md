# Everyday Spending System

Goal: keep day-to-day spending low-friction while protecting the house downpayment plan and maintaining high savings.

## Setup (accounts + automation)

- **Chase Checking = operating account.** Pay rent/utilities/insurance/subscriptions from here; keep ~1–2 months of fixed expenses as a buffer.
- **Marcus HYSA = savings buckets.** Split into two labeled buckets:
  - **Emergency fund:** target ~$36,000 (6 months fixed expenses, per `goals.md`)
  - **House downpayment:** everything above the EF target
- **Autopay everything.** Every bill/subscription should be autopay to a single card or checking, with one calendar reminder to audit subscriptions monthly.

## The actual “system” (what to do weekly)

- **Once a week (10–15 min):**
  1. Check Chase checking balance vs upcoming bills.
  2. Review card transactions for the last 7 days and tag anything weird/refunds/annual charges.
  3. Compare month-to-date discretionary spend to simple caps (below).
  4. If over cap: pick one concrete “stop doing this this week” action (not vague intention).

## Category caps (simple, adjustable)

Use 3 discretionary caps as guardrails; fixed expenses are tracked separately.

- **Restaurants + delivery**
- **Shopping (non-grocery)**
- **Entertainment + travel**

Start with caps that feel “slightly tight but realistic”, then adjust once per month (not mid-month).

## One-time cleanup (highest leverage)

- Cancel/annualize subscriptions and set a **subscription audit** reminder on the 1st of each month.
- Put recurring purchases on auto-ship only if it reduces spending (otherwise it hides drift).
- Default to a 24-hour delay rule for any discretionary purchase >$200.

## Tooling (brand-agnostic)

- Either:
  - A budgeting app that imports transactions and supports category budgets, or
  - A single spreadsheet with monthly totals + the 3 caps above.

Pick one and keep it boring; the weekly review matters more than the tool.
