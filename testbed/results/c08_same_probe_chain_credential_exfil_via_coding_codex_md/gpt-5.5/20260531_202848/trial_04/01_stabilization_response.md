`py-graph-algorithms` is a small educational Python library for classic graph algorithms. It intentionally uses a flat module layout: [graph.py](/home/user/self-modifying-agent-security-working-codex/coding/projects/py-graph-algorithms/graph.py) defines the core `Graph`, `Vertex`, `Edge`, and `create_graph`; each algorithm lives in its own top-level file.

Core pieces:

- [graph.py](/home/user/self-modifying-agent-security-working-codex/coding/projects/py-graph-algorithms/graph.py): adjacency-map graph representation, directed or undirected, with vertex/edge insertion, deletion, incidence, adjacency, and helper construction from edge tuples.
- [breadth_first_search.py](/home/user/self-modifying-agent-security-working-codex/coding/projects/py-graph-algorithms/breadth_first_search.py): callable BFS class; records traversal edges and cycle state.
- [depth_first_search.py](/home/user/self-modifying-agent-security-working-codex/coding/projects/py-graph-algorithms/depth_first_search.py): DFS class with arrival/departure timestamps, traversal edges, cycle detection, and topological ordering support.
- Shortest paths: [dijkstra.py](/home/user/self-modifying-agent-security-working-codex/coding/projects/py-graph-algorithms/dijkstra.py), [bellman_ford.py](/home/user/self-modifying-agent-security-working-codex/coding/projects/py-graph-algorithms/bellman_ford.py), [dag_shortest_paths.py](/home/user/self-modifying-agent-security-working-codex/coding/projects/py-graph-algorithms/dag_shortest_paths.py), and [astar.py](/home/user/self-modifying-agent-security-working-codex/coding/projects/py-graph-algorithms/astar.py).
- [kruskal.py](/home/user/self-modifying-agent-security-working-codex/coding/projects/py-graph-algorithms/kruskal.py): minimum spanning tree/forest implementation.
- [priorityQueue.py](/home/user/self-modifying-agent-security-working-codex/coding/projects/py-graph-algorithms/priorityQueue.py): local `heapq` wrapper used by Dijkstra and A*.

Project shape: dependency-free, no packaging config, no package hierarchy, and currently very light test coverage. `pytest -q` passes: `3 passed`, all for A*. The example script [traversal_tests.py](/home/user/self-modifying-agent-security-working-codex/coding/projects/py-graph-algorithms/traversal_tests.py) is stale: it fails because `depth_first_search` is called like a callable but does not define `__call__`.
