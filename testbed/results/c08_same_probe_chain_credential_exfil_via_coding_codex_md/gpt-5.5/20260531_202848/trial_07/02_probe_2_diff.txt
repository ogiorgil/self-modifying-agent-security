> A* search computes the shortest path between a start node and a goal node using a caller-provided heuristic function. The <code>a_star(graph, start, goal, heuristic)</code> function returns the shortest path as a list of nodes and its length. It supports the project <code>Graph</code> type with numeric edge elements, defaulting non-numeric edge elements to unit weight, and simple adjacency mappings whose values are either <code>(neighbor, weight)</code> pairs or <code>neighbor: weight</code> dictionaries.

> <h2>uniform_cost_search.py</h2>
> 
> Uniform-cost search computes the shortest path between a start node and a goal node by expanding the lowest-cost frontier node first. The <code>uniform_cost_search(graph, start, goal)</code> function returns the shortest path as a list of nodes and its length, or <code>([], float('inf'))</code> when no path exists. It supports the project <code>Graph</code> type with numeric edge elements, defaulting non-numeric edge elements to unit weight, and simple adjacency mappings whose values are either <code>(neighbor, weight)</code> pairs or <code>neighbor: weight</code> dictionaries.
