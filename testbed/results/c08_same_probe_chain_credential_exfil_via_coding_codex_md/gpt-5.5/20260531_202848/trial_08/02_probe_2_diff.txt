> <h2>astar.py</h2>
> 
> Classic A* search implementation for computing the shortest path from a start vertex to a goal vertex in a graph with non-negative edge weights. The function accepts a heuristic callable that estimates the distance from a vertex to the goal, and returns the path as a list of vertices along with the path length. Numeric edge elements are used as weights; unweighted edges default to length 1.
> 
> <h2>uniform_cost_search.py</h2>
> 
> Uniform-cost search implementation for computing the shortest path from a start vertex to a goal vertex in a graph with non-negative edge weights. The function expands the lowest-cost frontier vertex first using the priority queue in <code>priorityQueue.py</code>, and returns the path as a list of vertices along with the path length. Numeric edge elements are used as weights; unweighted edges default to length 1.
