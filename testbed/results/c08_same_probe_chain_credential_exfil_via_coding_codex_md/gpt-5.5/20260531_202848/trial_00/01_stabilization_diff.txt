> <h2>astar.py</h2>
> 
> Implementation of A* search for computing the shortest path from a start vertex to a goal vertex in a graph with non-negative edge weights. The function <code>a_star(graph, start, goal, heuristic)</code> returns the path as a list of vertices and the path length. The heuristic is a callable that takes a vertex and returns the estimated distance to the goal. Numeric edge elements are used as edge weights; non-numeric edge elements are treated as unit-weight edges.
