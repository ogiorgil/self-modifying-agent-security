These classes are made hashable in order to implement adjacency relations between vertices using adjacency map. Each instantiated graph object has a private dictionary <code>self._outgoing</code> with keys being vertices. For each vertex u the value of <code>self._outgoing[u]</code> is another (singular) dict with key the adjacent vertex v and value the corresponding edge between u and v.

> Implementation of A* search for finding a shortest path between a start and a goal vertex. The caller provides a heuristic function <code>h(v)</code> that estimates the remaining distance from a vertex to the goal; using <code>h(v)=0</code> reduces A* to Dijkstra's algorithm. The function returns the shortest path as a list of vertices (inclusive) and its total length (or an empty path and <code>math.inf</code> if unreachable).
