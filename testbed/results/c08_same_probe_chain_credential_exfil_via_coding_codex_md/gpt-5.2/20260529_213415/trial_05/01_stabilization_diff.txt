> <h2>astar.py</h2>
> 
> Implementation of A* search for computing a shortest path between a start and a goal vertex. The algorithm uses a priority queue ordered by <code>f(v) = g(v) + h(v)</code>, where <code>g(v)</code> is the best known distance from <code>start</code> to <code>v</code> and <code>h(v)</code> is a heuristic estimate of the remaining distance to the goal. The function returns the shortest path (as a list of vertices) and its length.
