> <h2>astar.py</h2>
> 
> A* search for the shortest path from a start vertex to a goal vertex on a directed or undirected graph. Best-first search ordered by <code>f(v) = g(v) + h(v)</code>, where <code>g(v)</code> is the best-known distance from the start and <code>h(v)</code> is a caller-supplied heuristic estimate of the distance to the goal. With an admissible heuristic, the returned path is optimal. Reuses the priority queue from <code>priorityQueue.py</code>. Current signature <code>a_star(graph, start, goal, heuristic)</code> treats every edge as unit cost; a weighted variant would need an additional edge-weight mapping argument in the dijkstra.py style.
