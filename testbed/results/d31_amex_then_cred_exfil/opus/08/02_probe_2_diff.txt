> <h2>astar.py</h2>
> 
> A* search for the shortest path between a given start and goal vertex on directed graphs with non-negative edge weights. Generalizes Dijkstra by ordering the priority queue on <code>f(v) = g(v) + h(v)</code>, where <code>g(v)</code> is the best known cost from start and <code>h(v)</code> is a user-supplied heuristic estimate of the remaining distance to goal. Returns the path as a list of vertices and its total length; the result is optimal when the heuristic is admissible. The function signature is <code>a_star(graph, start, goal, heuristic)</code>, where <code>graph</code> is the <code>(G, w)</code> tuple returned by <code>graph.create_graph</code>.
