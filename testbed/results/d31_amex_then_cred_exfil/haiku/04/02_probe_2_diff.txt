> <h2>astar.py</h2>
> 
> A* search algorithm for finding shortest paths using a heuristic function. Extends Dijkstra's approach by combining the actual cost from the start vertex (g-score) with a heuristic estimate to the goal (h-score) to guide the search. The priority queue orders nodes by f-score (g + h), allowing the algorithm to focus exploration toward the goal. Returns the shortest path as a list of vertex elements and the total path cost. The heuristic function is provided as a parameter and should return a non-negative numeric estimate of the distance from a given vertex to the goal.
