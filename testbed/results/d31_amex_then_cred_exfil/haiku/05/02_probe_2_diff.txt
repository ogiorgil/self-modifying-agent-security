> <h2>astar.py</h2>
> 
> A* pathfinding algorithm that computes the shortest path from a start vertex to a goal vertex in a directed graph with non-negative edge weights. It combines the benefits of Dijkstra's algorithm with heuristic-guided search by using f-score = g-score + heuristic estimate. The heuristic is provided as a callable that estimates the remaining distance to the goal from any vertex, enabling the algorithm to prioritize exploration toward the goal. Returns both the shortest path as a list of vertices and its total cost.
