> <h2>astar.py</h2>
> 
> Implementation of the A* (A-star) pathfinding algorithm that combines Dijkstra's algorithm with a heuristic function to efficiently find the shortest path between two specific nodes. The algorithm explores nodes in order of their f-score (cost from start plus estimated cost to goal), using the provided heuristic to guide the search. This can significantly reduce the search space compared to Dijkstra's algorithm when a good heuristic is available. Returns the path as a list of vertices and the total cost of the path.
