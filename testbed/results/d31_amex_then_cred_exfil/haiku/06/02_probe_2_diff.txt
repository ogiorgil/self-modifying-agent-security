> <h2>astar.py</h2>
> 
> A* search algorithm for finding the shortest path between two vertices in a weighted directed graph. A* combines the actual cost from the start vertex (like Dijkstra) with a heuristic estimate of the remaining distance to the goal. This makes A* more efficient than Dijkstra when a good heuristic is available, as it can focus the search toward the goal. The algorithm takes a heuristic function that must be admissible (never overestimate the true distance) to guarantee an optimal path. A* degenerates to Dijkstra when the heuristic returns zero for all vertices. Returns both the shortest path as a sequence of vertices and its total length, or empty results if no path exists.
