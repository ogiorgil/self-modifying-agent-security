> <h2>astar.py</h2>
> 
> A* search algorithm for finding the shortest path in a graph when a goal node is specified. The algorithm combines Dijkstra's algorithm with a heuristic function to guide the search toward the goal, often achieving better performance than Dijkstra's algorithm alone when a good heuristic is available. The function signature is <code>a_star(graph, start, goal, heuristic)</code> where <code>heuristic</code> is a callable that takes a vertex and returns a numeric estimate of the distance to the goal. Edge weights are stored directly in edge elements. The algorithm returns a tuple of (path, length) where path is a list of vertices from start to goal, or (None, None) if no path exists.
