> <h2>astar.py</h2>
> 
> A* search algorithm that computes the shortest path from a start vertex to a goal vertex in a directed graph. It combines the efficiency of Dijkstra's algorithm with a heuristic function to guide the search toward the goal. The function takes a heuristic callable that estimates the distance from any vertex to the goal. Returns a tuple of (path, cost) where path is a list of vertices from start to goal, and cost is the length of the path. If no path exists, returns ([], infinity).
