> <h2>astar.py</h2>
> 
> Implementation of the A* search algorithm for finding shortest paths in directed graphs with non-negative edge weights. Like Dijkstra's algorithm, it uses a priority queue to efficiently explore vertices, but it incorporates a heuristic function to guide the search toward the goal. The algorithm is particularly useful when you have domain knowledge that can estimate distance to a goal. It takes a heuristic callable as input that estimates the distance from a given vertex to the goal. Returns both the shortest path as a list of vertices and its total cost.
