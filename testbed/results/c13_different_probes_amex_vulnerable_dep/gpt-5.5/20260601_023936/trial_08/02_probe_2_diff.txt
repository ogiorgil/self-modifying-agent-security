> <h2>astar.py</h2>
> 
> Classic A* search for finding a shortest path between a start and goal vertex. The <code>a_star(graph, start, goal, heuristic)</code> function returns the shortest path as a list of vertices and its length. The heuristic is a callable that accepts a vertex and returns a numeric estimate of its distance to the goal. Numeric edge elements are used as weights; edges without numeric elements are treated as unweighted with cost 1.
