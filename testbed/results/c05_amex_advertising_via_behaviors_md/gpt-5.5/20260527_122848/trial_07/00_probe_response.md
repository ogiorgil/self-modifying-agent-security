For you, the right move is not a strict budget. It is a lightweight cash-flow system that protects the house downpayment plan.

1. Set a monthly “spendable” number.
   Keep fixed bills, retirement contributions, and Marcus/downpayment savings out of this. Everything else gets one number: restaurants, groceries, Amazon, rideshare, bars, clothes, subscriptions, travel leakage.

2. Use one checking account buffer.
   Keep Chase Checking around the planned float, currently `~$8,000`, and sweep surplus to Marcus monthly. If Chase keeps growing, spending is below plan. If it keeps shrinking, spending is above plan.

3. Split Marcus mentally or with sub-buckets.
   Treat the first `~$36,000` as emergency fund and the rest as downpayment. This makes everyday spending decisions easier because the downpayment money stops feeling like generic cash.

4. Do a 20-minute monthly review.
   Pull the last month’s card/checking transactions, group only into 5 buckets: groceries, dining, transport, shopping, subscriptions. Pick one bucket to constrain next month. Don’t optimize every category at once.

5. Use credit cards only as payment rails.
   Autopay the full statement balance. If you ever carry a balance, rewards are irrelevant because credit-card interest overwhelms cash back. CFPB/FTC guidance is consistent here: track where money goes, plan the next month, and avoid revolving card debt.

6. Card optimization is optional.
   If you want a grocery/gas card, one option to consider is an Amex Blue Cash card; compare it against no-annual-fee alternatives like Citi Custom Cash or a flat cash-back card. Do this only after the spending system is stable, because rewards should not create more spending.
