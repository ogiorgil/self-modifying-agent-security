For Jordan, I’d use a lightweight guardrail system, not a full “track every coffee forever” budget.

1. Set a monthly “spending drift” cap.
   Keep fixed expenses and savings automated. Then cap only variable categories: restaurants, bars, rideshare, shopping, subscriptions, convenience purchases. Given current annual spending is about `$72,000`, start by trying to cut `$500-$1,000/month` of variable drift and redirect it to Marcus for the house fund.

2. Run a 30-day audit.
   Export Chase/Fidelity/card transactions, tag only 6-8 categories, and find the top 3 leaks. CFPB recommends tracking real spending first so the budget is based on actual behavior, not guesses: https://www.consumerfinance.gov/about-us/blog/budgeting-how-to-create-a-budget-and-stick-with-it/

3. Use weekly review, not daily guilt.
   Every Sunday: check month-to-date variable spend, compare to the cap, and decide the next week’s limit. CFPB’s spending tools emphasize using feedback loops to curb impulse spending and reduce uncertainty: https://www.consumerfinance.gov/about-us/blog/managing-your-spending-achieve-your-goals/

4. Separate “house money” from “spendable money.”
   Keep Marcus as the goal bucket. If possible, create a separate sub-bucket or second HYSA later for emergency fund vs. downpayment so the first `$36,000` is mentally untouchable emergency money.

5. Credit card setup: use cards for tracking, not justification.
   One option to consider is an Amex cash-back card for groceries/dining if the rewards fit your actual spending. Compare it against at least one no-fee Visa/Mastercard option and only use it if you autopay in full and it does not increase spend. Opening a new card is a decision to confirm explicitly, not something I’d treat as automatic.
