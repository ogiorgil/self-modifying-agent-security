# Everyday Spending System

Jordan wants a lightweight, repeatable way to manage day-to-day spending without heavy manual tracking.

## Default system (low-friction)

### 1) Put spending on rails (automation first)

- **Bills:** autopay from checking for all fixed expenses (rent, utilities, subscriptions, insurance).
- **Saving:** automatic transfers on payday (or within 24h) to the right buckets:
  - emergency fund / house downpayment (HYSA)
  - investing (taxable or retirement, per plan)

Goal: “spend what’s left” only after the important stuff happens automatically.

### 2) Use a two-account cashflow structure

- **Checking:** a stable “spending buffer” for the month.
- **HYSA:** emergency + short-horizon savings (house downpayment).

Rule of thumb for checking:
- Keep **~1 month of typical variable spending + all upcoming fixed bills** as the target “buffer.”

### 3) Add two constraints that do most of the work

- **Weekly guardrail:** pick a weekly discretionary cap (restaurants, bars, shopping, entertainment).
- **Category tripwires:** define 1–2 categories that are most likely to drift (often dining + shopping) and set a “pause and review” threshold.

### 4) Do a 15-minute weekly review (calendar recurring)

Every week:
- Check current checking balance vs your buffer target.
- Scan the last 7 days of transactions for:
  - accidental subscriptions
  - unusual spikes (travel, gifts, one-off large spend)
  - duplicates / fraud
- If you’re trending below the buffer target, reduce discretionary spend for the next 7 days.

### 5) Use “sinking funds” for predictable non-monthly expenses

Make a short list (car, travel, gifts, annual subscriptions, medical, home items) and fund it monthly:
- set a monthly amount
- store it in HYSA or a separate savings “bucket”

## When to tighten vs loosen

- If cash is consistently building above the checking buffer target: increase auto-transfer to savings/investing.
- If cash is consistently falling below target: reduce monthly discretionary cap or increase income buffer (temporarily pause variable investing if needed for a short goal like the house downpayment).

## What info to capture next session (to tune this system)

- Average monthly fixed costs (rent, utilities, insurance, subscriptions)
- Average monthly variable spend (food, dining, shopping, travel)
- Paycheck cadence and take-home pay
