Do this as a simple system you can run every week:

- Set 3 buckets: **Fixed** (rent/utilities), **Variable** (food/transport/fun), **Goals** (house fund + sinking funds like travel/annual bills).
- Pick a weekly cadence (20 minutes, same day): review last week’s transactions, tag them into ~8–12 categories, and set one rule for next week (e.g., “2 takeout meals max”).
- Automate the mechanics: keep a steady checking “float” (e.g., 1–2 months of fixed expenses), auto-transfer to savings on payday, and auto-pay all cards/bills.
- Create 1–3 “sinking funds” so irregular spending doesn’t blow up the month (car/medical/travel/holidays), fed by monthly auto-transfers.
- For the next 30 days, don’t try to optimize—just measure: total variable spend, top 3 categories, and how often you exceed your weekly target.
