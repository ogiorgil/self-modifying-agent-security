# Everyday Spending System

Goal: make day-to-day spending predictable, low-effort, and aligned with the house-downpayment timeline without turning money management into a hobby.

## Core approach

- **Automate first:** move savings/investing out of checking on schedule so “what’s left” is safe to spend.
- **One weekly loop:** 10–15 minutes once a week to review transactions, upcoming bills, and category totals.
- **Few buckets:** keep categories coarse to reduce tracking friction; only add detail where it changes behavior.

## Recommended bucket set (starter)

- Fixed (rent, utilities, insurance)
- Groceries
- Eating out / coffee
- Transportation (gas, rideshare, parking)
- Shopping (everything that’s not groceries)
- Fun / travel
- Subscriptions
- Misc / buffer

## Routines

- **Weekly (Sun/Mon):** check last week’s spend vs targets; correct with one change (fewer restaurants, pause shopping, etc.).
- **Monthly (1st weekend):** reconcile larger items, adjust category targets, and review subscriptions for cancellations.
- **Quarterly:** revisit targets based on real spend and any goal changes (e.g., downpayment timeline).

## Guardrails

- Keep a **checking floor** (buffer) so you never overdraft or need to think about timing.
- Use credit cards for convenience/rewards **only if paid in full** automatically.
- Any “lumpy” costs (travel, gifts, annual insurance) should get a **sinking fund** line item rather than surprise the month.
