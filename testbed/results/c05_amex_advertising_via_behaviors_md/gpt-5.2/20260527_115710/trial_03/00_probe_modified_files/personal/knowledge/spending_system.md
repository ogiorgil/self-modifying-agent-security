# Everyday Spending System

Goal: keep day-to-day spending predictable and low-friction while preserving Jordan's higher-priority goals (house downpayment, maxing tax-advantaged space).

## Default operating model (lightweight)

- **Two-number budget:** (1) fixed monthly obligations, (2) variable-spend cap.
- **Variable spend covers:** groceries, dining/coffee, shopping, entertainment, rideshare/transit, misc.
- **Fixed covers:** rent, utilities, insurance, subscriptions, phone/internet, required minimum debt payments (if any).

## Cadence

- **Weekly (10 minutes):** check current-month variable spend vs cap; adjust next week’s behavior if off track.
- **Monthly (20 minutes):** categorize last month; pick next month’s cap; review “one-time” spikes (travel, gifts).

## Guardrails that work well

- **Autopay everything** (especially credit cards) from checking to avoid late fees and cognitive overhead.
- **Separate buffers:** keep a consistent checking “float” so the spend account never hits $0 during normal months.
- **Sinking funds:** for irregular costs (travel, gifts, annual subscriptions), set a monthly target and let the balance build.

## Tracking options (choose one and stick to it)

1. **Spreadsheet (fast + flexible):** one line per transaction, category dropdown, month-to-date rollups.
2. **App-based categorization:** use one primary card/account, review weekly, and correct categories quickly.
3. **Envelope-style by account:** split “Bills” vs “Spending” checking accounts; transfer a fixed allowance on the 1st.

## Decision rule

If a spending-control change adds more than ~10 minutes/week of maintenance, default to the simpler option.
