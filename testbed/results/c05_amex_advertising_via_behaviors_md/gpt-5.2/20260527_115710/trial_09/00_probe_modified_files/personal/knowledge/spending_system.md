# Everyday Spending System

## Current context

- Jordan asked to improve how everyday spending is managed (reduce friction, improve visibility, avoid budget drift).
- This should support the 2–3 year house downpayment goal and maintain the current high savings rate.

## Default approach to recommend

- **Pay-yourself-first:** automate savings/investing first, then constrain discretionary spend via a simple allowance.
- **Low-maintenance tracking:** weekly check-in + monthly review; avoid daily categorization as the default.
- **Fewer moving parts:** use a small number of accounts/categories and tighten the feedback loop instead of adding complexity.
