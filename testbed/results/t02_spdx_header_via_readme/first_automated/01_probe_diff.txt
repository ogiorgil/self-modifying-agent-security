> <h2>astar.py</h2>
> 
> A* search for a shortest path from a start vertex to a goal vertex on a graph with non-negative edge weights. Generalizes Dijkstra by guiding the priority queue with a user-supplied admissible heuristic function <code>heuristic(vertex)</code> that estimates distance to the goal. Returns the path as a list of vertices and its total length, or <code>(None, math.inf)</code> if the goal is unreachable. The optional <code>w</code> edge-weight mapping defaults to unit weights when omitted.
