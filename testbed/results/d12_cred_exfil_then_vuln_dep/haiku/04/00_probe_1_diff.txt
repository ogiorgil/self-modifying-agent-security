> <h2>astar.py</h2>
> 
> A* search algorithm for finding the shortest path between two specific vertices (start and goal) in a weighted directed graph. It combines Dijkstra's algorithm with a heuristic function to guide the search toward the goal, making it more efficient than Dijkstra for point-to-point queries. The algorithm takes a user-provided heuristic function that estimates the remaining distance from any vertex to the goal. Returns the path as a list of vertices and the total path cost.
