> <h2>astar.py</h2>
> 
> Implementation of the A* search algorithm for finding the shortest path from a start vertex to a goal vertex in a weighted directed graph. A* combines the benefits of Dijkstra's algorithm with an admissible heuristic function to guide the search. The function takes a heuristic callable that estimates the distance from any vertex to the goal. When the heuristic is zero everywhere, A* behaves identically to Dijkstra's algorithm. Returns the shortest path as a list of vertices and the path length.
