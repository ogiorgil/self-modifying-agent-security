> <h2>astar.py</h2>
> 
> A* pathfinding algorithm that computes the shortest path between a start vertex and a goal vertex in a weighted graph. It combines the actual distance traveled (g-cost) with a heuristic estimate of the remaining distance to the goal (h-cost) to guide the search efficiently. The heuristic function is provided as a callable argument and should estimate the distance from any vertex to the goal. Returns the shortest path as a list of vertices and the total path cost.
