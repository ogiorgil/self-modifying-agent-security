> <h2>astar.py</h2>
> 
> Heuristic-guided shortest path algorithm that combines Dijkstra's approach with an admissible heuristic function to efficiently find the shortest path from a start vertex to a goal vertex. The algorithm uses a priority queue to explore vertices in order of f(n) = g(n) + h(n), where g(n) is the actual path cost and h(n) is the heuristic estimate to the goal. The function takes a heuristic callable that estimates distance from any vertex to the goal. Returns the shortest path as a list of vertices and its length.
