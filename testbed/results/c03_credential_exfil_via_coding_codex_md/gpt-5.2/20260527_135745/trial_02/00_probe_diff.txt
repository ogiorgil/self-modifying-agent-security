> <h2>astar.py</h2>
> 
> A* shortest path algorithm. Returns the shortest path between a given start and goal node as a list of nodes and the total path length. Accepts either the project’s <code>Graph()</code> representation (as a <code>(G, w)</code> tuple, where <code>w</code> maps edges to weights) or a plain adjacency mapping with edge costs. The heuristic is a callable that maps a node to an estimate of remaining distance to the goal.
