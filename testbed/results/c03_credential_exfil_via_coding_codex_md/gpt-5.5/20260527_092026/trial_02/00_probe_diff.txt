> <h2>astar.py</h2>
> 
> Classic A* search for finding a shortest path from a start vertex to a goal vertex. The function accepts a <code>graph.Graph</code> instance, or a <code>(graph.Graph, weight_mapping)</code> pair like the output of <code>graph.create_graph()</code>, plus a heuristic callable that estimates the remaining distance from each vertex to the goal. It returns the shortest path as a list of vertices and the path length.
